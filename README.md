# B站视频下载器

一个美观易用的B站视频下载工具，支持将视频转换为MP3音频格式。

## 功能特点

- 🎬 支持批量下载用户空间所有视频
- 🎵 自动转换为MP3音频格式
- 🖥️ 美观的Web界面
- ⚡ 支持单个视频下载
- 🔧 可配置下载路径、代理等设置
- 📦 打包为独立exe，无需配置环境

## 使用方法

### 方式一：直接运行exe（推荐）

1. 下载 `release` 目录中的发布包
2. 解压后双击 `B站视频下载器.exe`
3. 浏览器会自动打开Web界面

### 方式二：Python源码运行

```bash
# 安装依赖
pip install flask

# 安装外部工具
pip install yt-dlp
# 下载ffmpeg并添加到PATH

# 运行
python app.py
```

## 界面说明

1. **批量下载** - 输入B站用户空间地址或用户ID，获取所有视频列表
2. **单个下载** - 输入视频链接或BV号，下载单个视频
3. **下载管理** - 查看下载进度和历史
4. **设置** - 配置下载路径、代理、音频质量等

## 注意事项

- 首次运行可能需要等待几秒钟
- B站有反爬虫机制，频繁请求可能被限制
- 如遇网络问题，请在设置中配置代理
- 下载的文件默认保存在 `downloads` 目录

## 技术栈

- **后端**: Python + Flask
- **前端**: HTML + CSS + JavaScript
- **下载工具**: yt-dlp
- **音视频转换**: ffmpeg

## 目录结构

```
├── app.py              # Flask主程序
├── main.py             # 打包入口
├── templates/          # HTML模板
├── static/             # 静态资源
│   ├── css/           # 样式文件
│   └── js/            # JavaScript
├── dist/              # 打包输出
└── release/           # 发布包
```
